# Refactor Search Bar Interaction

## Why

优化搜索体验，提供更沉浸和流畅的搜索交互。通过动效和推荐标签引导用户更高效地使用搜索和标签筛选功能。

## What Changes

### 1. 搜索框动效
- **默认状态**: 显示为常规搜索框。
- **激活状态**: 点击搜索框时，搜索栏平滑展开/放大（可覆盖部分头部或向下延展）。
- **动画**: 使用 `matchedGeometryEffect` 或弹簧动画实现平滑过渡。

### 2. 视觉样式优化
- Input Placeholder: 低对比度文本 "Search... (type # for tags)"。
- 增加视觉层级，激活时给予更强的阴影或背景区分。

### 3. 推荐标签面板
- 在搜索框激活时，在输入框下方（或作为下拉面板）展示“推荐标签”。
- 推荐内容：
    - `allTags` 中的前几个热门标签。
    - 点击标签自动填充到搜索框（例如 "#Design"）。

### 交互流程
1. 用户点击搜索框 → `isSearching = true`
2. 搜索框带动画展开，下方淡入显示标签云
3. 用户输入文字或点击标签
4. 点击空白处或取消按钮 → `isSearching = false`，恢复原状

## Out of Scope
- 搜索历史记录
- 复杂的富文本搜索语法
